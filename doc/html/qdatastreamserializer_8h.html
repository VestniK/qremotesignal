#define qDataStreamSerializer   qrs::GlobalSerializer<qrs::QDataStreamSerializer>::instance()

Pointer to single global instance of QDataStreamSerializer class which uses latest version of the QDataStream protocol available in Qt library used to build this library.

In general it's not recommended to use this macro since th version of the underlying protocol used by serializer is depends on environment at build time and may be unpredictable.
